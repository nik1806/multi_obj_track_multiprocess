# multi_obj_track_multiprocess
Faster implementation of multiple object tracking by leveraging multiple core 

### Dependencies
1. python (ver >= 3.6)
2. Opnecv (ver >= 3)
3. argparser

### Execution

Use following command for help
```
python multi_pro_tracking.py -h
```
or simply 
```
python multi_pro_tracking.py
```
